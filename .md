figma-to-angular/

 ├── output/
 │   └── figma-raw.json
 ├── .env
 └── package.json

 | Step | Script                              | Purpose                                                                                          | Output                                             |
| ---- | ----------------------------------- | ------------------------------------------------------------------------------------------------ | -------------------------------------------------- |
| 1    | `scripts/fetch-figma.js`            | Fetch raw Figma JSON from the Figma URL using your token                                         | `figma-raw.json`                                   |
| 2    | `scripts/normalize-figma.js`        | Normalize Figma nodes (frames, groups, components) into a **clean schema**                       | `normalized-figma.json`                            |
| 3    | `scripts/generate-ui-data.js`       | Convert normalized Figma JSON → **UI Schema + UI Models + Mock Data**                            | `ui-schema.json`, `ui-models/*.ts`, `ui-mock/*.ts` |
| 4    | `scripts/generate-design-tokens.js` | Extract **colors, fonts, spacing, border-radius** from Figma JSON → central **design-tokens.ts** | `design-tokens/design-tokens.ts`                   |
| 5    | `scripts/generate-angular-ui.js`    | Generate **Angular components** from UI schema & mock data, apply responsive SCSS + tokens       | `components/*.component.ts/.html/.scss`            |
| 6    | `scripts/generate-pages.js`         | Assemble components into **Angular pages**, create **routes**                                    | `pages/*.page.ts/.html/.scss`, `app.routes.ts`     |


# 1️⃣ Fetch raw Figma JSON
node scripts/fetch-figma.js

# 2️⃣ Normalize Figma JSON
node scripts/normalize-figma.js

# 3️⃣ Generate UI data (schema, models, mock)
node scripts/generate-ui-data.js

# 4️⃣ Generate design tokens
node scripts/generate-design-tokens.js

# 5️⃣ Generate Angular components (responsive + tokens)
node scripts/generate-angular-ui.js

# 6️⃣ Generate pages + routing
node scripts/generate-pages.js
